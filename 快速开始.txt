🚀 第一次使用 (只需 3 步):

步骤 1: 提交配置文件到 GitHub
    cd /path/to/VxKex
    git add .github/ *.md
    git commit -m "Add automated build and release workflow"
    git push origin main

步骤 2: 创建版本标签
    git tag v1.1.3.1428
    git push origin v1.1.3.1428

步骤 3: 等待构建完成 (约 10-15 分钟)
    访问: https://github.com/YOUR_USERNAME/VxKex/actions
    查看构建进度

🎉 完成! 在 Releases 页面即可下载安装程序!

🔐 可选: 配置代码签名

1. 购买代码签名证书 (DigiCert, Sectigo 等)
2. 导出为 PFX 格式并转换为 Base64
3. 在 GitHub Settings → Secrets 中添加:
   - CERTIFICATE_BASE64
   - CERTIFICATE_PASSWORD
4. 参考 .github/workflows/README.md 修改工作流
